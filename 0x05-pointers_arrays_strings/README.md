## *POINTERS*

* Pointers are" variable-like" structures or references that holds a memory address to another varible, arrays.

* Pointers act as a memory address and they dont store any values.

## *ARRAYS*

* Array is the collection of similar type of data type.

## *Strings*

* String is a collection of characters used to represent data.
